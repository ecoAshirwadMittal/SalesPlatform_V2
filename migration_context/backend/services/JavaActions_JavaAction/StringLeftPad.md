# Java Action: StringLeftPad

> Pads a string on the left to a certain length. value : the original value amount: the desired length of the resulting string. fillCharacter: the character to pad with. (or space if empty) For example StringLeftPad("hello", 8, "-") returns "---hello" StringLeftPad("hello", 2, "-") returns "hello"

**Returns:** `String`

## Parameters

| Name | Type | Required |
|---|---|---|
| `value` | String | ✅ |
| `amount` | Integer | ✅ |
| `fillCharacter` | String | ✅ |
