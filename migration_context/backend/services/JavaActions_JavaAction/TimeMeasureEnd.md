# Java Action: TimeMeasureEnd

> End timing something, and print the result to the log. - TimerName. Should correspond to the TimerName used with TimeMeasureStart. - LogLevel. The loglevel used to print the result. - The message to be printed in the log.

**Returns:** `Integer`

## Parameters

| Name | Type | Required |
|---|---|---|
| `TimerName` | String | ✅ |
| `Loglevel` | Enumeration | ✅ |
| `message` | String | ✅ |
