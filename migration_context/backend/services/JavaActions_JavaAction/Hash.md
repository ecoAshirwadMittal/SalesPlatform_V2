# Java Action: Hash

> Hashes a value using the SHA-256 hash algorithm. - value : the value to hash - length : the desired length of the hash. Returns a SHA-256 hash of 'value', with length 'length'

**Returns:** `String`

## Parameters

| Name | Type | Required |
|---|---|---|
| `value` | String | ✅ |
| `length` | Integer | ✅ |
