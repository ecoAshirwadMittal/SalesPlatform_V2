# Java Action: copyAttributes

> Copies all common primitive attributes from source to target, which are not necessarily of the same type. This is useful to, for example, translate database object into view objects. Note that no automatic type conversion is done.

**Returns:** `Boolean`

## Parameters

| Name | Type | Required |
|---|---|---|
| `source` | ParameterizedEntity | ✅ |
| `target` | ParameterizedEntity | ✅ |
