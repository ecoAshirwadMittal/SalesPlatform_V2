# Java Action: TimeMeasureStart

> Start timing something, and print the result to the log. - TimerName. Should correspond to the TimerName used in TimeMeasureEnd. Note that multiple timers can run at once. Existing timers can be restarted using this function as well.

**Returns:** `Boolean`

## Parameters

| Name | Type | Required |
|---|---|---|
| `TimerName` | String | ✅ |
