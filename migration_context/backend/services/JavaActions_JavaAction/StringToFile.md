# Java Action: StringToFile

> Stores a string into the provided FileDocument, using the specified encoding. Note that destination will be committed.

**Returns:** `Boolean`

## Parameters

| Name | Type | Required |
|---|---|---|
| `value` | String | ✅ |
| `destination` | ConcreteEntity | ✅ |
| `encoding` | Enumeration | ✅ |
