# Java Action: DecryptPrivateKey

> This java action takes a passphrase as well as an p8 private key file as an input parameter and decrypts the private key.

**Returns:** `String`

## Parameters

| Name | Type | Required |
|---|---|---|
| `PrivateKey` | ConcreteEntity | ✅ |
| `Passphrase` | String | ✅ |
