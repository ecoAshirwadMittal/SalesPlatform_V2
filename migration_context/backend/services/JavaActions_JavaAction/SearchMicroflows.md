# Java Action: SearchMicroflows

> Get all microflows without parameters where CompleteName contains SearchFor parameter. Note: Manually scheduled microflow cannot have any parameters But when scheduled programmatically parameters are allowed (eq QueuedActionParameters)

**Returns:** `List`

## Parameters

| Name | Type | Required |
|---|---|---|
| `SearchFor` | String | ✅ |
| `ParametersAllowed` | Boolean | ✅ |
