# Java Action: HTMLEncode

> Encodes a string to HTML Entities, so that they can be displayed in the browser without breaking any layout. This is useful for special widgets which allow HTML to be rendered properly, including special characters as '<' and '&'. For example '<' will be encoded as '&lt;' and '&' will be encoded as '&amp;' Returns the encoded string.

**Returns:** `String`

## Parameters

| Name | Type | Required |
|---|---|---|
| `value` | String | ✅ |
