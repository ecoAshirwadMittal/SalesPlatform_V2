# Java Action: YearsBetween

> Calculates the number of years between two dates. - dateTime : the original (oldest) dateTime - compareDate: the second date. If EMPTY, the current datetime will be used. Effectively this means that the age of the dateTime is calculated.

**Returns:** `Integer`

## Parameters

| Name | Type | Required |
|---|---|---|
| `dateTime` | DateTime | ✅ |
| `compareDate` | DateTime | ✅ |
