# Java Action: SetStringParameterDeeplink

> Creates a deeplink configuration using DeepLinkname as identifier. Any existing configurations with this identifier will be deleted.

**Returns:** `Void`

## Parameters

| Name | Type | Required |
|---|---|---|
| `Microflow` | Unknown | ✅ |
| `Description` | String | ✅ |
| `DeepLinkName` | String | ✅ |
| `ForceLogin` | Boolean | ✅ |
| `KeepAsHomeDeeplink` | Boolean | ✅ |
