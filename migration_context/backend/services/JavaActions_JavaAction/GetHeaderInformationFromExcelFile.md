# Java Action: GetHeaderInformationFromExcelFile

> Synchronise all excel columns and add/update/remove the existing columns from the template in the parameter. The return value is irrelevant and will be always true

**Returns:** `Boolean`

## Parameters

| Name | Type | Required |
|---|---|---|
| `TemplateObject` | ConcreteEntity | ✅ |
| `ExcelFileDocument` | ConcreteEntity | ✅ |
