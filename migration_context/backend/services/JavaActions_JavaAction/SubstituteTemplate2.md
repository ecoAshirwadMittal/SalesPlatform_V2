# Java Action: SubstituteTemplate2

> Identical to SubstituteTemplate, but adds an datetimeformat argument DateTimeFormat identifies a format string which is applied to date/time based attributes. Can be left empty. Defaults to "EEE dd MMM yyyy, HH:mm"

**Returns:** `String`

## Parameters

| Name | Type | Required |
|---|---|---|
| `template` | String | ✅ |
| `substitute` | ParameterizedEntity | ✅ |
| `useHTMLEncoding` | Boolean | ✅ |
| `datetimeformat` | String | ✅ |
