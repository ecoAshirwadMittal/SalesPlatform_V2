# Java Action: refreshClassByObject

> Refreshes a certain domain object type in the client. Useful to enforce a datagrid to refresh for example. - instance : This object is used to identify the type of objects that need to be refreshed. For example passing $currentUser will refresh all System.Account's.

**Returns:** `Boolean`

## Parameters

| Name | Type | Required |
|---|---|---|
| `instance` | ParameterizedEntity | ✅ |
