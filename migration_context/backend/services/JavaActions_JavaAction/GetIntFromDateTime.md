# Java Action: GetIntFromDateTime

> Converts a datetime to an integer based on the selector used. Selectors available are: - year (returns f. ex. 1980) - month (returns 1-12) - day (returns 1-31)

**Returns:** `Integer`

## Parameters

| Name | Type | Required |
|---|---|---|
| `dateObj` | DateTime | ✅ |
| `selectorObj` | Enumeration | ✅ |
