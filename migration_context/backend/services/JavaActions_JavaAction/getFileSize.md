# Java Action: getFileSize

> Returns the filesize of a file document in bytes. From version 2.3 on, this function can be used in cloud environments as well. NOTE: before 2.1, this functioned returned the size in kilobytes, although this documentation mentioned bytes

**Returns:** `Integer`

## Parameters

| Name | Type | Required |
|---|---|---|
| `document` | ConcreteEntity | ✅ |
