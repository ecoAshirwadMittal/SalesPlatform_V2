# Java Action: StringRightPad

> Pads a string on the right to a certain length. value : the original value amount: the desired length of the resulting string. fillCharacter: the character to pad with. (or space if empty) For example StringRightPad("hello", 8, "-") returns "hello---" StringLeftpad("hello", 2, "-") returns "hello"

**Returns:** `String`

## Parameters

| Name | Type | Required |
|---|---|---|
| `value` | String | ✅ |
| `amount` | Integer | ✅ |
| `fillCharacter` | String | ✅ |
