# Java Action: ExecuteDeeplink

> Executes a Pendlinglink, i.e. executes the corressponding microflow. Returns true if a link was executed, false otherwise. If this action returns true, no further actions should be taken in the current context.

**Returns:** `Boolean`

## Parameters

| Name | Type | Required |
|---|---|---|
| `pendinglink` | ConcreteEntity | ✅ |
