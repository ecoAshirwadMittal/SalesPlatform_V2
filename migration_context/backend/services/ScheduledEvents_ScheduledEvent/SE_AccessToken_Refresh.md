# Scheduled Event: SE_AccessToken_Refresh

> This scheduled event refreshes the access token that is used to secure access to the Document Generation Cloud service. If your app is running in the Mendix cloud, this scheduled event needs to be enabled.

## Configuration

| Property | Value |
|---|---|
| **Status** | ✅ Enabled |
| **Microflow** | `DocumentGeneration.SE_AccessToken_Refresh` |
| **Interval** | Every 1 Minute(s) |
| **Start Time** | Tue May 23 2023 10:22:36 GMT+0000 (Coordinated Universal Time) |
| **Time Zone** | UTC |
