# Scheduled Event: SE_LogCleanUp

> This action will cleanup the loglines. The logs entries will be archived for the amount of days specified in the configuration, the log entries that reached the limit will be deleted.

## Configuration

| Property | Value |
|---|---|
| **Status** | ❌ Disabled |
| **Microflow** | `SAML20.SE_LogCleanUp` |
| **Interval** | Every 1 Day(s) |
| **Start Time** | Wed Sep 12 2012 04:00:00 GMT+0000 (Coordinated Universal Time) |
| **Time Zone** | Server |
