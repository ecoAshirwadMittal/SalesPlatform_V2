# Scheduled Event: SE_DocumentRequest_Cleanup

> This scheduled event removes expired DocumentRequest objects after a configured offset in days (using constant @DocumentGeneration.RequestCleanupOffsetInDays, the default value is 7 days).

## Configuration

| Property | Value |
|---|---|
| **Status** | ✅ Enabled |
| **Microflow** | `DocumentGeneration.SE_DocumentRequest_Cleanup` |
| **Interval** | Every 1 Minute(s) |
| **Start Time** | Tue Nov 22 2022 08:47:41 GMT+0000 (Coordinated Universal Time) |
| **Time Zone** | UTC |
