# Constant: LoginLocation

> If login is required, this constant defines where to use can enter his credentials. If empty, the default location in the theme folder is used: login.html The original deeplink location will be appended to the login location when the login location ends with a '='. For example, in case of Mendix SSO: 'https://login.mendix.com/openid/login?continuation=' Default: empty

| Property | Value |
|---|---|
| **Type** | `String` |
| **Default Value** | `(empty)` |
| **Exposed to Client** | ❌ |
