# Constant: MergeMultiplePdfs_MaxAtOnce

> Restricted to 10 files at once for Mendix Cloud v4 compatibility. If you need to merge more than 10 files increase the number here. Setting the value to <=0 means unlimited. Note: We very strongly recommend to not increase the number for applications running in Mendix Cloud v4.

| Property | Value |
|---|---|
| **Type** | `Integer` |
| **Default Value** | `10` |
| **Exposed to Client** | ❌ |
