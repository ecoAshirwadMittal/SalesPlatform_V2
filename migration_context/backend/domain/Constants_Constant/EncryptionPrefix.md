# Constant: EncryptionPrefix

> Prefix for stored encrypted values. This value is used to determine whether the value has been already encrypted. The prefix must be either {AES}, {AES2} or {AES3}. Encryption is only supported if the prefix is {AES3}. It is not recommended to change this value.

| Property | Value |
|---|---|
| **Type** | `String` |
| **Default Value** | `{AES3}` |
| **Exposed to Client** | ❌ |
