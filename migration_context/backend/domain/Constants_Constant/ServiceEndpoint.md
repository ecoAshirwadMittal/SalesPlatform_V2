# Constant: ServiceEndpoint

> The endpoint of the PDF Document Generation service. Only applicable when using the 'Cloud' or 'Private' service type. Default: https://docgen.home.mendix.com (public PDF document generation service on Mendix Public Platform)

| Property | Value |
|---|---|
| **Type** | `String` |
| **Default Value** | `https://docgen.home.mendix.com` |
| **Exposed to Client** | ❌ |
