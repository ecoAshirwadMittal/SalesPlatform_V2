# Constant: DefaultLogoutPage

> It is recommended to remove the sign out button, but if you choose to keep the sign out button the user will be redirected to a page. After signing out the user will be redirected to this location.

| Property | Value |
|---|---|
| **Type** | `String` |
| **Default Value** | `/SSO/` |
| **Exposed to Client** | ❌ |
