### SPKB-2294 — Bid Ranking | Verify Admin Ranking Page Filters by Year/Week and Ranking Calculation

- **Status**: Done
- **Priority**: P2
- **Labels**: `9.2.0`
- **Linked Issues**: SPKB-1968

**Description:**
Preconditions:
- User is logged in as Admin.
- Auction data exists for multiple years and weeks.
- Participants have EB values assigned.
Test Steps:
- Navigate to Admin → Ranking Management page.
- Select an Auction Year from the year dropdown.
- Select an Auction Week from the week dropdown.
- Click Search to load the ranking list.
- Verify rankings load correctly for the selected year and week.
- Click Recalculate Ranking.
- After recalculation completes, open a participant’s ranking detail.
Expected Results:
- Ranking list displays results only for the selected Year + Week combination.
- Recalculate Ranking button is visible and functional for Admin.
- Final ranking score includes EB (Extra Bonus) values:
Final Score = Base Score + EB Score.
- Rankings reorder if EB changes affect the score.
- No blank page, errors, or unexpected behavior.
