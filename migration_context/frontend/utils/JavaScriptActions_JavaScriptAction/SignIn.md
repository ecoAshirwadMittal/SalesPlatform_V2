# JavaScript Action: SignIn

> Tries to login using a username and password. Returns an HTTP response status code, for example: - 200 when the login succeeds - 401 when the entered username or password is incorrect - 0 when the network connection is unavailable

**Returns:** `Integer`

## Parameters

| Name | Type | Required |
|---|---|---|
| `Username` | String | ✅ |
| `Password` | String | ✅ |
