# JavaScript Action: DraftEmail

> Start drafting an email in the platform specified email client. This might work differently for each user depending on their platform and local configuration.

**Returns:** `Boolean`

## Parameters

| Name | Type | Required |
|---|---|---|
| `Recipient` | String | ✅ |
| `Cc` | String | ✅ |
| `Bcc` | String | ✅ |
| `Subject` | String | ✅ |
| `Body` | String | ✅ |
