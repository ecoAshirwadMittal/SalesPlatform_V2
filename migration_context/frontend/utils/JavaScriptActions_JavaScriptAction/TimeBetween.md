# JavaScript Action: TimeBetween

> The TimeBetween function calculates the difference between the input Date and times in milliseconds, seconds, minutes, hours or days, depending on the unitOfTime.

**Returns:** `Integer`

## Parameters

| Name | Type | Required |
|---|---|---|
| `startDate` | DateTime | ✅ |
| `endDate` | DateTime | ✅ |
| `unitOfTime` | String | ✅ |
