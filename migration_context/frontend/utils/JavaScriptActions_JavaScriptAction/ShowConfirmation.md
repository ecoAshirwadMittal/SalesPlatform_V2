# JavaScript Action: ShowConfirmation

> Shows a confirmation dialog during the execution of a nanoflow, to make perform actions based on the user input.

**Returns:** `Boolean`

## Parameters

| Name | Type | Required |
|---|---|---|
| `titleCaption` | String | ✅ |
| `Question` | String | ✅ |
| `CancelButtonCaption` | String | ✅ |
| `ProceedButtonCaption` | String | ✅ |
